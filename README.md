# Frontend Angular — Étude Réussie

```bash
npm install
npm start
```

Routes principales :

- `/login`
- `/eleve/tableau-de-bord`
- `/tuteur/tableau-de-bord`
- `/parent/tableau-de-bord`
- `/admin/tableau-de-bord`
- `/super-user/utilisateurs`

Le frontend fonctionne en mode démonstration avec `localStorage`. Le bouton `Réinitialiser` remet les données de départ.
